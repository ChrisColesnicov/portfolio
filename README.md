# 🌐 Mon Portfolio Personnel

Un portfolio moderne pour présenter mes compétences et projets en tant que développeur web.

# 🚀 Description

Ce portfolio a pour objectif de mettre en avant :
	•	Mes compétences techniques 🛠️
	•	Mes projets réalisés 🎯
	•	Mon parcours professionnel 👨‍💻
	•	Un formulaire pour me contacter facilement 📬

Créé avec amour ❤️ et des technologies modernes comme React, Next.js, Tailwind CSS, et des animations avec Framer Motion.

# ✨ Fonctionnalités principales
	•	🌟 Page d’accueil : Présentation personnelle avec photo et tagline.
	•	📂 Projets : Une galerie interactive pour explorer mes réalisations.
	•	📞 Contact : Un formulaire simple pour me joindre.
	•	📱 Responsive Design : Adapté à tous les écrans (mobile, tablette, desktop).
	•	🎥 Animations : Transitions fluides et modernes avec Framer Motion.

# 🛠️ Technologies utilisées
	•	⚡ Framework Frontend : Next.js
	•	🎨 UI Styling : Tailwind CSS
	•	🎥 Animations : Framer Motion
	•	🗂️ Gestion des états : React Context (ou autre selon les besoins).
	•	☁️ Déploiement : Vercel


# 🧑‍💻 Installation et démarrage du projet

1.	Clone le repo :

```
git clone https://github.com/ton-utilisateur/nom-du-repo.git
```
```
cd nom-du-repo
```

2.	Installe les dépendances :
```
npm install
```
ou
```
yarn install
```

3.	Lance l’application en mode développement :
```
npm run dev
```
ou
```
yarn dev
```
4.	Ouvre http://localhost:3000 dans ton navigateur. 🚀
